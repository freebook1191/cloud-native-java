# bootcamp
take your first steps with Cloud Foundry and Spring Boot
[클라우드 네이티브 자바] 예제 파일 다운로드



이 책에서 설명하는 예제 파일은 원서 깃허브 페이지에서 내려받으실 수 있습니다. 

https://github.com/cloud-native-java/



챕터별 예제 파일은 다음과 같습니다.



2장, 부트캠프: 스프링 부트와 클라우드 파운드리 소개 - https://github.com/cloud-native-java/bootcamp

3장, 12요소 애플리케이션 설정 - https://github.com/cloud-native-java/configuration

4장, 테스트 - https://github.com/cloud-native-java/testing

5장, 애플리케이션 마이그레이션 - https://github.com/cloud-native-java/forklifting

6장, REST API - https://github.com/cloud-native-java/rest

7장, 라우팅 - https://github.com/cloud-native-java/routing

8장, 엣지 서비스 - https://github.com/cloud-native-java/edge

9장, 데이터 관리 - https://github.com/cloud-native-java/data

10장, 메시징 - https://github.com/cloud-native-java/messaging

11장, 배치 처리와 태스크 - https://github.com/cloud-native-java/integration

12장, 데이터 통합 - https://github.com/cloud-native-java/data-integration

13장, 관측 가능한 시스템 - https://github.com/cloud-native-java/operations

14장, 서비스 브로커 - https://github.com/cloud-native-java/service-brokers

15장, 지속적 전달 - https://github.com/cloud-native-java/continuous-delivery

부록 A, 자바 EE와 스프링 부트 https://github.com/cloud-native-java/ee



이 책의 원서 깃허브 리포지토리에는 원저자가 컨퍼런스에서 시연에 사용했던 프로젝트들도 함께 포함되어 있기 때문에, 이 책에서 사용된 예제 코드가 포함된 프로젝트를 찾는 데 약간의 불편함이 있습니다. 



(위에 열거한 깃허브 리포 인덱스에 나온) 챕터별 루트 폴더를 기준으로 예제 코드 최상단에 있는 package 키워드로 명시된 패키지 이름으로 폴더를 찾고, class 키워드로 명시된 클래스 파일 이름을 찾으면 예제별 소스 파일을 조금 더 쉽게 찾으실 수 있습니다.
